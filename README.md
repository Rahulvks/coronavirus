# coronavirus
Python web scraper for https://www.worldometers.info/coronavirus/ 

#### Installation:

`git clone https://github.com/flamebarke/coronavirus`

`cd coronavirus`

`pip3 install -r requirements.txt`

#### Usage: 

`python3 ./coronavirus.py`
